# اسکریپت دمو – دستیار فروش تلگرام

## پیش‌نیاز
```bash
pip install -r requirements.txt
cp .env.example .env
# TELEGRAM_BOT_TOKEN و اختیاری OPENAI_API_KEY را تنظیم کنید
```

## اجرای سریع فالوآپ (۶۰ ثانیه / ۱۲۰ ثانیه)
```bash
set FOLLOWUP_1H_SECONDS=60
set DISCOUNT_2D_SECONDS=120
python -m app.main
```

## سناریوهای دمو

### ۱. شروع مکالمه
- کاربر: `/start`
- انتظار: پیام خوش‌آمد فارسی

### ۲. درخواست ناقص
- کاربر: `یه لپ‌تاپ می‌خوام`
- انتظار: ۲ سوال (بودجه + کاربرد)

### ۳. تکمیل نیاز
- کاربر: `برای برنامه‌نویسی، بودجه ۸۰ میلیون`
- انتظار: حداقل ۳ پیشنهاد با دلیل و نقطه ضعف

### ۴. سوال پیگیری از حافظه
- کاربر: `اون دومی قیمتش چقدره؟`
- انتظار: پاسخ بدون جستجوی مجدد کامل

### ۵. مقایسه
- کاربر: `اولی و دومی رو مقایسه کن`
- انتظار: جدول/متن مقایسه

### ۶. تغییر بودجه
- کاربر: `بودجه‌ام رو به ۳۰ میلیون تغییر دادم`
- انتظار: پیشنهادهای جدید

### ۷. عکس
- ارسال عکس با کپشن `هدفون بی‌سیم`
- انتظار: محصولات مشابه

### ۸. لینک
- ارسال URL محصول
- انتظار: محصولات مشابه یا درخواست توضیح بیشتر

### ۹. فالوآپ ۱ ساعته (دمو: ۶۰ ثانیه)
- بعد از پیشنهاد، ۶۰ ثانیه سکوت
- انتظار: پیام پیگیری مودبانه

### ۱۰. تخفیف ۲ روزه (دمو: ۱۲۰ ثانیه)
- بدون `/mark_purchased`
- انتظار: پیام با کد SALES10

### ۱۱. ثبت خرید
- `/mark_purchased`
- انتظار: تأیید خرید، بدون فالوآپ تخفیف

### ۱۲. ریست
- `/reset`
- انتظار: شروع مکالمه از نو **و پاک شدن حافظه LangGraph**

## راهنمای ضبط ویدئوی دمو (تحویل پروژه)

### تنظیمات `.env` برای ضبط سریع
```bash
TELEGRAM_BOT_TOKEN=...
GROQ_API_KEY=...          # اختیاری؛ Tool Agent + polish
USE_LLM=true
FOLLOWUP_1H_SECONDS=60
DISCOUNT_2D_SECONDS=120
VECTOR_STORE_BACKEND=keyword
```

### اجرای ربات
```bash
python -m app.main
```

### چک‌لیست ویدئو (حدود ۵–۸ دقیقه)

| # | نمایش در ویدئو | نکته |
|---|----------------|------|
| 1 | `/start` | پیام خوش‌آمد |
| 2 | `یه لپ‌تاپ می‌خوام` | سوال بودجه/کاربرد |
| 3 | `برای برنامه‌نویسی، بودجه ۸۰ میلیون` | **۳+ پیشنهاد + عکس محصول** |
| 4 | `اون دومی قیمتش چقدره؟` | پاسخ از حافظه |
| 5 | `اولی و دومی رو مقایسه کن` | مقایسه |
| 6 | `بودجه‌ام رو به ۳۰ میلیون تغییر دادم` | پیشنهاد جدید |
| 7 | عکس + کپشن `هدفون بی‌سیم` | مشابه‌یابی |
| 8 | لینک محصول | مشابه‌یابی |
| 9 | ۶۰ ثانیه سکوت | فالوآپ ۱ ساعته |
| 10 | ۱۲۰ ثانیه سکوت | تخفیف SALES10 |
| 11 | `/mark_purchased` | ثبت خرید |
