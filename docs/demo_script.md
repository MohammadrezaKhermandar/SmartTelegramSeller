# اسکریپت دمو – دستیار فروش تلگرام

## پیش‌نیاز

```bash
pip install -r requirements.txt
cp .env.example .env
# TELEGRAM_BOT_TOKEN را تنظیم کنید
```

## اجرای ربات (فالوآپ کوتاه برای ضبط ویدئو)

**Windows (PowerShell / CMD):**
```bash
set FOLLOWUP_1H_SECONDS=60
set DISCOUNT_2D_SECONDS=120
python -m app.main
```

**Linux / macOS:**
```bash
export FOLLOWUP_1H_SECONDS=60
export DISCOUNT_2D_SECONDS=120
python -m app.main
```

> فقط **یک** نمونه ربات اجرا کنید. چند instance همزمان → خطای `Conflict` در تلگرام.

## سناریوی اصلی دمو (ترتیب پیشنهادی)

| # | پیام / اقدام کاربر | انتظار |
|---|-------------------|--------|
| 1 | `/start` | پیام خوش‌آمد فارسی |
| 2 | `یه لپ‌تاپ می‌خوام` | ۱–۲ سوال تکمیلی (بودجه / کاربرد) |
| 3 | `بودجه ۱۰۰ میلیون برای کار دانشجویی و برنامه‌نویسی` | حداقل ۳ پیشنهاد لپ‌تاپ + عکس محصول |
| 4 | `اون دومی رمش چقدره؟` | پاسخ از **حافظه** (بدون جستجوی کامل مجدد) |
| 5 | `بودجه شد ۳۰ میلیون` | دسته **لپ‌تاپ** حفظ می‌شود؛ بودجه به‌روز می‌شود. اگر در این بودجه لپ‌تاپ موجود نباشد، پیام «محصول دقیق پیدا نشد» — **نه** محصول نامرتبط |
| 6 | `بودجه شد ۵۰ میلیون` | پیشنهادهای جدید، همچنان از دسته لپ‌تاپ |
| 7 | `گزینه اول و سوم رو مقایسه کن` | متن/جدول مقایسه دو محصول پیشنهادی |
| 8 | ارسال **URL** یک محصول | محصولات مشابه یا درخواست توضیح بیشتر |
| 9 | ارسال **عکس** محصول (با کپشن اختیاری) | محصولات مشابه |
| 10 | **۶۰ ثانیه سکوت** | فالوآپ ۱ ساعته (با `FOLLOWUP_1H_SECONDS=60`) |
| 11 | **۱۲۰ ثانیه سکوت** (بدون `/mark_purchased`) | پیام تخفیف ۲ روزه با کد SALES10 |

### دستورات تکمیلی (اختیاری در انتهای ویدئو)

| دستور | انتظار |
|--------|--------|
| `/mark_purchased` | تأیید خرید؛ بدون فالوآپ تخفیف بعدی |
| `/reset` | شروع مکالمه از نو + پاک شدن حافظه LangGraph |
| `/my_products` | نمایش آخرین پیشنهادها |
| `/help` | راهنمای دستورات |

## نکات مهم برای ضبط

### بودجه و دسته
- تغییر بودجه (`بودجه شد …`) دسته قبلی را **جایگزین نمی‌کند**.
- اگر در بودجه/دسته محصولی نباشد، ربات باید بگوید دقیق پیدا نشد — نه اینکه کیف/هدفون و غیره نشان دهد.

### LLM / API
- اگر Groq/OpenAI خطای **403** یا **429** داد: کلید معتبر بگذارید یا `USE_LLM=false` برای fallback آفلاین.
- بدون LLM، `hybrid_search` + keyword RAG همچنان کار می‌کند.

### env پیشنهادی برای ضبط
```env
TELEGRAM_BOT_TOKEN=...
GROQ_API_KEY=...          # اختیاری
USE_LLM=true
FOLLOWUP_1H_SECONDS=60
DISCOUNT_2D_SECONDS=120
VECTOR_STORE_BACKEND=keyword
```
